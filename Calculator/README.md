# Calculator

It is a programme written in Kotlin designed to an implementation of Handy Calculator for macOS.

## Describing the code

It provides different methods and classes in order to make calculations.

### Methods in MainActivity.kt

- buNumberEvent: adds a new digit to the calculator's screen
- buOpEvent: adds the operation to the calculator's screen
- buEqualEvent: does the operation
- buPercent: calculates percentage of a number
- buClean: cleans the number on the calculator's screen
